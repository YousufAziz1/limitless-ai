## 🚀 The Vision

In India, **250 million students** live in rural areas with unreliable or non-existent internet. While city students use ChatGPT and online coaching, rural students are left behind.

**Limitless AI is not just another wrapper.** It is a complete, standalone, **"Grandma-Level Easy"** Offline AI Operating System. 

We took the complexity of local LLMs (Python, Node.js, CMD, Ollama, Virtual Environments) and packaged it into a **single, beautiful 1-click Windows Installer (`.exe`)**.

---

## 🏆 Why Limitless AI Stands Out (For Judges)

### 1. Zero-Friction Onboarding ("Grandma-Level Easy")
Most local AI projects require users to open the terminal, install Python, run `pip install`, and pull models manually. **Limitless AI changes the game.**
- Users download `LimitlessAISetup.exe`.
- They double click it.
- **That's it.** The app launches a beautiful cinematic onboarding sequence that silently installs the AI Engine (Ollama), downloads Google's Gemma model, and starts the bundled FastAPI backend. No terminal. No coding.

### 2. Standalone Compiled Backend
The Python FastAPI backend is compiled into a standalone executable using **PyInstaller**. The user’s machine does not need Python installed. Everything is bundled securely.

### 3. Beautiful, Cinematic UI
Built with React 18, Vite, Tailwind CSS, and Framer Motion. Every button, card, and layout is highly polished with micro-interactions, gradients, and a cyberpunk-inspired dynamic mesh background. It looks and feels like a YC-backed startup product.

### 4. Truly Offline
Once the initial 1-click setup is done, Limitless AI runs **100% offline**. No internet required. Zero latency. Complete privacy.

## 🏗️ Architecture

```
Limitless-ai/
├── frontend/             # Electron App + React 18 + Vite
│   ├── electron/         # Electron Main Process (Silent Installer, Backend Spawner)
│   ├── src/              # React Components, Pages, Zustand Store
│   └── package.json      # electron-builder config
│
├── backend/              # Standalone Python FastAPI Core
│   ├── main.py           # Core logic, SSE streaming, Ollama routing
│   ├── build.bat         # PyInstaller compilation script
│   └── dist/main.exe     # The compiled executable (No Python required!)
```

### The 1-Click Pipeline
1. `LimitlessAISetup.exe` installs the Electron app.
2. Electron `main.cjs` launches the React UI.
3. React UI triggers the setup process.
4. Electron silently downloads/installs Ollama.
5. Electron pulls the `gemma4` model.
6. Electron spawns the bundled `backend/dist/main.exe` (FastAPI).
7. User starts learning seamlessly.

## 💻 Local Installation (For Judges)

For the hackathon submission, the 1-click Electron `.exe` installer is temporarily bypassed to ensure smooth local testing. Please follow these simple steps to run Limitless AI natively on your machine:

### 1. Install Ollama
Download and install [Ollama](https://ollama.com). Then, open your terminal and run:
```bash
ollama pull gemma4
```

### 2. Start the Backend
Open a terminal in the `backend` folder and run the local FastAPI server:
```bash
cd backend
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\python main.py
```
*(The backend runs on `http://localhost:8000`)*

### 3. Start the Frontend
Open a new terminal in the `frontend` folder and run the React app:
```bash
cd frontend
pnpm install
pnpm dev
```
*(Open `http://localhost:5173` in your browser)*

## 🛠️ Developer Setup (Manual Build)

If you want to build the project from source:

### 1. Build the Standalone Backend
```bash
cd backend
python -m venv venv
venv\Scripts\pip install -r requirements.txt
.\build.bat
# This creates backend/dist/main.exe
```

### 2. Run / Build the Frontend
```bash
cd frontend
pnpm install

# Run in dev mode
pnpm run electron:dev

# Build the Setup .exe
pnpm run electron:build
# The setup file will be in frontend/release/
```
